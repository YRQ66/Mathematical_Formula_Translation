## Requiremets
- Apple Silicon (M1) version
```
pip install -r requirements_all.txt
```
torch는 Apple Silicon 버전으로 맞추어 있기 때문에 torch는 따로 install 
- Windows
```
pip install -r requirements.txt
&& 디바이스에 맞는 torch 버전 다운
```
